# 使用

# 触发方式

  • 指令 + @user，如： / 爬 @小 Q
  • 指令 + qq 号，如：/ 爬 123456
  • 指令 + 自己，如：/ 爬 自己
  • 指令 + 图片，如：/ 爬 [图片]

前三种触发方式会使用目标 qq 的头像作为图片
