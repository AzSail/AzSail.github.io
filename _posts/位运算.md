###### 136. Single Number

* 交换律：a ^ b ^ c <=> a ^ c ^ b
* 任何数于0异或为任何数 0 ^ n => n
* 相同的数异或为0: n ^ n => 0

###### 1758. Minimum Changes To Make Alternating Binary String

![1758](https://mioe-xyz.oss-cn-shanghai.aliyuncs.com/usr/uploads/2022/11/2630183447.png)

```cpp
// 1 & 1 = 1

// 1 & 0 = 0

// 0 & 0 = 0

int minOperations(string s) {
        char a[2] = {'0', '1'};
        int ans = 0, t = 0;
        for(int i = 0; i < s.size(); i++) {
            if(a[i & 1] == s[i]) ans++;
            else t++;
        }
        return min(ans, t);
    }
```
